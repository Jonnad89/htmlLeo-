Crear un "Gestor de turnos"

consigna:

Tenes que crear una aplicación que permita anotar el nombre de un cliente
y la hora de su turno
Estos turnos deben guardarse en el navegador para que, aunque se corte
la luz o se cierre la pestaña, la agenda siga ahí.

Lo que debe tener la app: 
HTML: Un input para el nombre, otro para la hora y un botón de Agendar
LocalStorage: Al cargar la página, debe buscar si ya habia turnos guardados

DOM: Una lista donde aparezcan los turnos con un botón que diga "Atendido"
