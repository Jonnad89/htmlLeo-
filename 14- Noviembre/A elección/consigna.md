Crear una pequeña aplicación web (HTML, CSS, y JavaScript) sobre cualquier tema que te interese (ej: calculador de propinas, galería de imágenes, generador de citas al azar, validador de formularios, etc.). La aplicación debe ser funcional e interactiva.


Requisito JS/DOM	Concepto a Practicar
1.	Manejo de Inputs	
Debe leer datos de un campo de entrada (input type="text", number o select) y usar su valor (.value).

2.	Manejo de Eventos	
Debe usar al menos un listener de evento distinto a un clic de botón (ej: 'input', 'change', 'submit', 'mouseover').

3.	Lógica Condicional	
Debe usar una estructura if/else o if/else if/else para tomar una decisión en función de la entrada del usuario.

4.	Manipulación de Clases	
Debe modificar los estilos de al menos un elemento usando .classList.add() / .remove() o .classList.toggle() (ej: para 
mostrar un error, cambiar el tema, o resaltar un resultado).

5.	Manipulación de Salida	
Debe escribir o actualizar dinámicamente el contenido de un elemento HTML usando .textContent o .innerHTML para mostrar el resultado de la lógica.
