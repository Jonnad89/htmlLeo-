El objetivo es construir una tarjeta de presentación simple con una sección de controles

Cambiar el Tema: Implementar un interruptor (toggle) para alternar la clase de toda la página entre light-mode y dark-mode.

Color Dinámico: Usar un input de tipo color para cambiar dinámicamente el color de acento de la tarjeta (por ejemplo, el borde o el título).

Tamaño de Fuente: Usar un input de tipo range (slider) para ajustar el tamaño de fuente del texto de la tarjeta en tiempo real.
